OnlyPost Setup Instructions

Please follow these instructions carefully and in order. This process should take about 20-30 minutes from start to finish.


Saving the OnlyPost and Scripts Folders onto Your Desktop

  1. Open the email I sent you containing the blue OnlyPost folder.
  2. Right click the blue OnlyPost folder.
  3. Click 'Save Attachment'
  4. Click the drop down menu to the right of 'Where:' and select 'Desktop' (See image below step 6 for reference).
  5. Click the save button.
  6. Right click the blue Scripts folder and save it to your desktop like you did with the OnlyPost folder above.



Using the Terminal

  1. Click the magnifying glass icon located at the top right of your screen (or hit command+space).
  2. Type ‘Terminal’ into the Spotlight Search bar (do not include quotation marks).
  3. Double click the 'Terminal' result with the black icon next to it to open the terminal.
  4. Right-click the Terminal icon in your Dock, hover over “Options" and then click "Keep in Dock”.
  5. Click the red button in the upper left area of the terminal window (this is how you will close the terminal window from here on out).

Updating Your macOS

  1. Minimize your browser window and all other windows so that you can see the blue Scripts folder on your desktop (or search for 'Scripts' on your mac via Finder).
  2. Double click the Scripts folder so that it opens.
  3. Right click on the file titled osx_upgrade, hover over "Open with" option, and click Terminal.
  4. If a window pops up that says: "osx_upgrade cannot be opened because it is from an unidentified developer. macOS cannot verify that this app is free from malware." please press the open button. If this doesn't happen then please move on to the next step.
  5. If your operating system is up to date enough to use the OnlyPost script you will see a message that says "Everything is up to date!". If your operating system is not up to date enough to use the OnlyPost script you will be prompted to update it. Please do so if prompted.
  6. Close the terminal window when the script is done executing and you see the "[Process completed]" message at the bottom of your terminal screen.

Installing Requirements

  1. Minimize your browser window and all other windows so that you can see the blue Scripts folder on your desktop (or search for 'Scripts' on your mac via Finder).
  2. Double click the Scripts folder so that it opens.
  3. Right click on the file titled devToolsInstall, hover over "Open with" option, and click Terminal.
  4. If a window pops up that says: "devToolsInstall cannot be opened because it is from an unidentified developer. macOS cannot verify that this app is free from malware." please press the open button. If this doesn't happen then please move on to the next step.
  5. Enter your macOS user password when prompted.
  6. The terminal will display a list of what will be installed. Hit return (the enter button) when you are asked to do so.
  7. Close the terminal window when the requirements are finished installing and you see the "[Process completed]" message (this process might take a few minutes).
